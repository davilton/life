#Conaway's Game of Life

Dependencies:
```
$	gem install rspec
$	gem install factory_girl
$	gem install curses
```
To run:
```
$	./run_life.rb
```
	OR
```
$	./run_life.rb [size_of_board] [number_of_generations] [initial_conditions.txt]
```
To run tests:
from the root directory of the project, run:
```
$	rspec
```
